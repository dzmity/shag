Написать следующущие классы:
Класс Subject - Enum - список изучаемых предметов в университете
Класс Student:
Поля:
    Имя
	Фамилия
	Возраст (15 - 99)
	Курс (1 - 5)
    Делает ли домашки к каждому занятию (boolean)
	Список оценок (List<Integer>)
    Список изучаемых предметов (List<Subject>)
    Список предметов, по которым будет зачет в этом семестре (List<Subject>)
    Список предметов, по которым будет экзамен в этом семестре (List<Subject>)

Конструкторы:
	По умолчанию
	С первыми 5-ю полями из списка выше
Методы:
	Геттеры/Сеттеры
	(при установке некорректноого значения для курса - бросать кастомный (самописный) эксепшн типа Runtime)
	(при установке некорректноого значения для возраста - бросать кастомный (самописный) эксепшн типа Runtime)
	toString()

Класс HeadTeacher:
Поля:
	Имя
	Фамилия
	Список студентов (List<Student>)
	Список студентов на отчисление (List<Student>)
Конструкторы:
	По умолчанию
	Со всеми полями кроме Списка студентов на отчисление
Методы:
	Геттеры/Сеттеры
	* toString()

------------------------------------------------------------------------------------
В main() создать 4 студентов и создать завуча (использовать ранее созданных студентов)

Добавить в классе завуча ряд методов:

Метод печатает в консоль имена и фамилии своих студентов

Метод который принимает название предмета и выводит студентов (имя и фамилию) изучающих его.

Метод который принимает название предмета и выводит в консоль студентов (имя и фамилию) у кого по нему зачет.

Метод который принимает название предмета и выводит в консоль студентов (имя и фамилию) у кого по нему экзамен.

Метод checkBeforeSession() который печатает в консоль фамилии и среднюю оценку своих студентов,
также если оценка < 4 и он НЕ делает домашки, то студент добавляется в список на отчисление (если его там еще не было).
