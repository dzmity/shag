                        Task1
1. Создайте класс описывающий продукты питания, например FoodStuff
2. Класс необходимо написать c закрытыми (private) полями, например :
foodName, typeOfFood, productName, brandName, price, content, …
(минимум 3 строкового типа (String), и 3 целочисленного типа (int))
3. Создать по два открытых (public) метода для каждого поля, которые позволят устанавливать (set) и возвращать (get)
значения этих полей для каждого из экземпляров данного класса.
Причем обязательным условием будет не позволить присвоение следующих значений:
- 0 (нуль);
- отрицательное значение;
- пустая строка;
4. Реализуйте конструктор по умолчанию и конструктор для инициализации данных;
5. В методе main создать (new) 3 объекта этого класса, заполнить его поля используя методы
и вывести в консоль данные через запятую;

                        Task3
Описать базовый класс MainString (Строка). Обязательные поля класса:
■ массив символов;
■ значение типа int хранит длину строки в символах.

Реализовать обязательные методы следующего назначения:
■ конструктор без параметров;
■ конструктор, принимающий в качестве параметра строковый литерал;
■ конструктор, принимающий в качестве параметра символ;
■ метод получения длины строки;
■ метод очистки строки (делает строку пустой);
■ метод поиска символа в строке.
