                                Switch , case, break, default
Открыть презентацию по лекции 2 - слайды начиная с 24
Для упрощения реализации алгоритма множественного выбора в Java добавлен оператор switch.
Оператор анализирует выражение в скобках и передает управление одному из сценариев (case). Далее программа выполнит
весь код, находящийся в сценарии до конца оператора switch. Значения вариантов в case не должны повторяться.
Ключевое слово default может использоваться в операторе switch в случае, если ни один сценариев не был выполнен.
В блоке switch может присутствовать только одно ключевое слово default.Порядок расположения блоков
case и default не имеет значения, но принято располагать блоки case по возрастанию значения, а default – в самом конце.
