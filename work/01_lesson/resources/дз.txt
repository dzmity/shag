                                        Задание №1
Объявить и проинициализировать по 10 переменных для каждого примитивного типа (можно только для int, double, boolean) на любую тему.
Например (эту тему можно продолжать!):

    Для boolean:
        Представьте, что Вы проходите «Детектор лжи», Вы обязаны отвечать только true(истина) или false(ложь):
        // Вы родились в январе?
        boolean bornInJanuary = true;
        // Вы знакомы с президентом?
        boolean knowPresident = false;
        // а хотели бы познакомиться?
        boolean wantToKnowPresident = false;
        //Читали ли Вы «Война и Мир»?
        boolean readWarAndPeace= true;
        и т.д. …

    Для int:
        Представьте, что Вы приехали за покупками в Европу
        У Вас есть список того что нужно купить и сколько.
        Поэтому Вам нужны переменные, которые будут созданы для хранения этого количества:

        int numberOfCoffeeNescafeGoldBlend750g = 3; // кофе
        int numberOfTiresGoodYearR17 = 4; // покрышки
        и т.д. …

    Для double:
        Продолжая тему покупок – задайте цену для всех покупок.
        double costOfCoffeeNescafeGold = 7.34 // 7,34 евро
        double costOfTireBridgestoneR17 = 80.5 // цена за одну
        и т.д. …


                                        Задание №2
Попытайтесь на основании ваших переменных составить любой алгоритм подсчета количества или суммы.
Необходимо чтобы все данные и результаты печатались в консоль т.е. используйте команду System.out.println()
Например, зная количество и цену товаров можно посчитать сумму затрат:

double price = costOfCoffeeNescafeGold * numberOfCoffeeNescafeGoldBlend750g + costOfTireBridgestoneR17* numberOfTiresBridgestoneR17;
System.out.println(“Примерная сумма затрат составит “ + price);

Любой алгоритм приветствуется!
