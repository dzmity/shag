
                    Что такое программа.

Программа — это набор (список) команд. Сначала исполняется первая команда, затем вторая, третья, и так далее.
Когда все команды исполнены, программа завершается.
Программы, написанные на языке Java, исполняет JVM (Java Virtual Machine – виртуальная машина Java).
JVM — это специальная программа, которая умеет исполнять программы, написанные на языке Java.

                Принципы написания кода на Java

1 - В языке программирования Java каждую команду принято писать с новой строчки.
В конце команды ставится точка с запятой.

2 - Программа не может состоять только из команд.
В языке программирования Java команда не может быть сама по себе, она – часть функции (в Java функции называют методами).
А метод – это часть класса. Или, другими словами, класс делится на методы, а методы на команды.
На примере строительства - Класс – это многоквартирный дом, функция/метод – это квартира, а команда – это комната.

3 - Программы на языке Java состоят из классов. Классов может быть десятки тысяч. Минимальная программа – один класс.
Для каждого класса заводится отдельный файл, имя которого совпадает с именем класса.
Внутри файлов содержится код (текст) на языке программирования Java.
Обычно код класса состоит из «имени класса» и «тела класса». Тело класса помещается в фигурные скобочки.
Тело класса может содержать переменные (их еще называют данные класса) и методы (функции класса).
Минимальная программа должна состоять минимум из одного класса, который должен содержать минимум один метод/функцию,
с которого начинается выполнение программы. Такой метод должен иметь имя main.
Класс, с которого начинается программа, может иметь любое имя, но метод main, с которого начинает выполняться программа,
всегда имеет один и тот же вид
Тело метода состоит из команд.
Можно даже сказать, что метод – это команды, объединенные в группу, которой дали имя (имя метода). И так и так будет верно.

								
								Переменные

Переменные – это специальная штука для хранения данных. Все данные в Java хранятся с помощью переменных.
Больше всего переменная по смыслу напоминает коробку.
Каждая переменная в Java имеет три важных свойства: тип, имя и значение.
1 Имя используется для того, чтобы отличать одну переменную от другой. Это как надпись на коробке.
2 Тип переменной определяет тип значений/данных, которые в ней можно хранить.
В коробке для торта храним торт,в коробке для обуви – туфли, и тд.
3 Значение – это некий объект, данные или информация, которая хранится в переменной.								

Каждый объект в языке Java имеет свой тип. Например, могут быть такие типы данных:
«целое число», «дробное число», «текст», «тип Кот», «тип Дом» и т.д.
У переменной тоже есть свой тип. Переменная может хранить значения только того же типа, что и она сама.
