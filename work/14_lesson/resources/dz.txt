Создать класс RegistrationForm для регистрации пользователя на белорусском сайте для получения почтовых отправлений:

- логин
- пароль
- имя
- фамиля
- возраст
- серия и номер паспорта (поле String passport)
- область (создайте соотвествующий ENUM)
- город
- улица
- номер дома
- номер квартиры
- почтовый индекс
Все поля классов приватные, конструктор по умолчанию, геттеры и сеттеры.
Переопределите метод toString() так, чтобы вместо секьюрных данных (пароль, паспорт) указывались *****

Создайте класс LoginDB, который представляет собой своеобразную базу данных уже созданных в системе логинов.
Класс должен содежать массив из 10 логинов (придумайте любые значения и проинициализируйте ими массив ) и статический метод,
проверяющий, есть ли переданный в качестве параметра метода логин уже в вашей "бд".
Проверка должна быть независима от регистра ("NagibatoR777" и "nAgibator777" считаем одинаковым логином).
Напоминаю, что строки сравниваются не на == , а на equals()

Создайте аналогичный класс PassportDB, который представляет собой своеобразную базу данных уже "использованных" при регистрации паспортов.
Класс должен содежать массив из 10 паспортов (придумайте любые значения и проинициализируйте ими массив ) и статический метод,
проверяющий, есть ли переданный в качестве параметра метода паспорт уже в вашей "бд".
Проверка должна быть независима от регистра ("mc456789" и "MC456789" считаем одинаковым паспортом).

Создайте 4 класса эксепшенов (не RuntimeException), каждую с 1 конструктором , содержащим сообщение об ошибке как параметр
LoginAlreadyExistException 
PassportAlreadyExistException
InvalidPostIndexException
NotEnoughtAgeException

Методы по установке значений (set-еры) логина, паспорта, почтового индекса и возраста должны выбрасывать указанные выше эксепшены в случаях 
1 - если такой логин уже есть в вашей "бд" - LoginAlreadyExistException
2 - если такой паспорт уже есть в вашей "бд" - PassportAlreadyExistException
3 - если почтовый индекс не соответствует формату (длинее или короче чем нужно) (пример валидного - 222310) - InvalidPostIndexException
4 - если пользователю меньше 18 лет - NotEnoughtAgeException

Создать в классе Runner (в методе main) 5 валидных объектов данного класса и 4 невалидных - по 1 на каждый возможный тип эксепшена.
Вывести на экран toString() для каждого созданного объекта.
