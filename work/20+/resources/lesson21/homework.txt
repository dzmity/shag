Создать класс ListManager и в нем следующие методы:

Метод создает лист строк и сортирует его по длине слов. Класс компаратор создать как отдельный top-level класс (отдельный файл).

Метод создает лист строк и сортирует его по первой букве слова. Класс компаратор создать как static nested в этом же классе.

Метод создает лист строк и сортирует его по последней букве слова. Класс компаратор создать как inner класс (не статик).

Метод создает лист строк и сортирует его по длине слов в обратном порядке. Класс компаратор создать как local класс (в методе).

Метод создает лист строк и сортирует его по первой букве в обратном порядке. Класс компаратор создать как анонимный класс.

* (по желанию) Метод создает лист строк и сортирует его по по последней букве слова в обратном порядке. Класс создать с помощью lambda функции.
