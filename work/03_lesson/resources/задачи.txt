                            1 тернарный оператор
Написать программу, которая будет считывать введенный вами текст.
Если это целое число - программа должна выводить фразу - "Вы ввели число = " и введенное число в одну строку.
Если это не целое число - программа должна выводить фразу - "Ошибка. Вы ввели не целое число"
Подсказка: необходимо использовать тернанрный оператор.

                            3  if else if
В отдельном методе написать программу, которая будет считывать введенное вами целое число
(случай, когда вводится не целое число в расчет не берем (для упрощения)) и в зависимости от его значения
- выводить на экран строку "Число меньше 5", если параметр метода меньше 5,

- выводить строку "Число равно 5", если параметр метода равен 5.

Даны три целых числа. Найти среднее из них (то есть число, расположенное между наименьшим и наибольшим).

                            8 if else if
Ввести с клавиатуры три целых числа. Определить, имеется ли среди них хотя бы одна пара равных между собой чисел.
Если такая пара существует, вывести на экран числа через пробел.
Если все три числа равны между собой, то вывести все три.

Примеры:
а) при вводе чисел 1 2 2 получим вывод 2 2
б) при вводе чисел 2 2 2 получим вывод 2 2 2

Пользователь сам вводит два целых числа. Программа задаёт вопрос: «результат умножения первого числа на второе = ?».
Пользователь должен ввести ответ и увидеть на экране правильно он ответил или нет.
Если нет — показать еще и правильный результат.
